整体二分

引入

在信奥赛中，有一部分题目可以使用二分的办法来解决。但是当这种题目有多次询问且我们每次查询都直接二分可能导致 ==TLE== （超时），就会用到整体二分。整体二分的主体思路就是把多个查询一起解决。（所以这是一个==离线算法==）

可以使用整体二分解决的题目需要满足以下性质：

  1. 询问的答案具有可二分性
  2. 修改对判定答案的贡献互相独立，修改之间互不影响效果
  3. 修改如果对判定答案有贡献，则贡献为一确定的与判定标准无关的值
  4. 贡献满足交换律，结合律，具有可加性
  5. 题目允许使用离线算法

解释

记 $[l,r]$ 为答案的值域，$[L,R]$ 为答案的定义域（为区间询问的左右边界）。（也就是说求答案时仅考虑下标在区间 $[L,R]$ 内的操作和询问，这其中询问的答案在 $[l,r]$ 内）

  • 我们首先把所有操作 按时间顺序 （按时间顺序的主要原因是带有修改操作的整体二分）存入数组中，然后开始分治（由于会将操作分组后分别进行，与二分的折半特征相比更像是分治）。
  • 在每一层分治中，利用数据结构（常见的是树状数组，支持 $O(logn)$ 区间查询的同时允许 $O(nlogn)$ 重构，注意重构不能使用memset）统计当前查询的答案和值域中点 $mid$ 之间的关系（树状数组支持区间查询是为了限制定义域 $[L,R]$）。
  • 根据查询出来的答案和 $mid$ 间的关系（小于等于 $mid$ 和大于 $mid$）将当前处理的操作序列分为 $ql$ 和 $qr$ 两份，并分别递归处理。
  • 当 $l=r$ 时，找到答案，记录答案并返回即可。

需要注意的是，在整体二分过程中，若当前处理的值域为 $[l,r]$，则此时最终答案范围不在 $[l,r]$ 的询问会在其他时候处理。另外，由于整体二分的时间复杂度与值域的大小直接相关，大部分题目往往会需要==离散化==操作。

过程

先来看一个简单的、询问中没有区间的例子：

查询全局第 k 小

题 1 在一个数列中查询第 $k$ 小的数。

当然可以直接排序．但是这里我们使用二分值域的方法：可以用数据结构记录每个大小范围内有多少个数，然后用二分法猜测，利用数据结构检验。

例如：

有一组数据：$a[10] = {9, 3, 2, 7, 4, 3, 1, 9, 7, 2}$

显然这组数据的值域为 $[1, 9]$，然后比如说我们查询全局第 $k=8$ 小：

首先计算当前值域中点 $mid$ 是 $5$，然后我们通过一定方式查询数组中在左值域 $[l,mid]$ 的数字个数 $t$，得到 $t=6$ （本例是在全局查询因此直接统计即可，如果询问带有区间限制的就需要使用树状数组），发现 $k > t$，因此可以判定我们要查询的结果不在左值域 $[1, 5]$ 中，因此我们继续往右值域也就是 $[6, 9]$ 中查询。

但是递归查询右值域时有一个特殊的点在于需要修改 $k$，因为上一步中左值域有 $6$ 个数字了，因此下一步应该需要在右值域中查询第 $2$ 小，也就是 $k-=t$。往左值域递归就不需要修改 $k$。

按照这种方式继续二分值域，最终到达递归边界 $l=r$ 时，$l$ 就是当前查询的答案。

题 2 在一个数列中多次查询第 $k$ 小的数。

可以按照上述二分值域的方法回答每个询问；但是，也可以把所有的询问放在一起整体二分。

先考虑二分的本质：假设要猜一个 $[l,r]$ 之间的数，猜测之后会知道是猜大了，猜小了还是刚好。当然可以从 $l$ 枚举到 $r$，但更优秀的方法是二分：猜测答案是 $m = \lfloor\frac{l + r}{2}\rfloor$，然后去验证 $m$ 的正确性，再调整边界。这样做每次询问的复杂度为 $O(\log n)$，若询问次数为 $q$，则时间复杂度为 $O(q\log n)$。

回过头来，对于当前的所有询问，可以去猜测所有询问的答案都是 $mid$，然后去依次验证每个询问的答案应该是小于等于 $mid$ 的还是大于 $mid$ 的，并将询问分为两个部分（不大于/大于），对于每个部分继续二分。注意：如果一个询问的答案是大于 $mid$ 的，则在将其划至右侧前需更新它的 $k$，即，如果当前数列中小于等于 $mid$ 的数有 $t$ 个，则将询问划分后实际是在右区间询问第 $k - t$ 小数。如果一个部分的 $l = r$ 了，则结束这个部分的二分。利用线段树的相关知识，我们每次将整个答案可能在的区间 $[1,n]$（假设已经离散化）划分成了若干个部分，这样的划分共进行了 $O(\log n)$ 次，一次划分会将整个操作序列操作一次。若对整个序列进行操作，并支持对应的查询的时间复杂度为 $O(T)$，则整体二分的时间复杂度为 $O(T\log n)$．

struct query {
  int k, id;  // 这个询问的 k, 这个询问的编号
};

int ans[MAXN], a[MAXN];  // ans[i] 表示编号为i的询问的答案，a 为原数列
int tmp[MAXN], cnt[MAXN];  // 离散化后，记录对应的值及其计数（假设在输入时已经处理好）

// 返回原数列中值域在 [l,r] 中的数的个数
int check(int l, int r) {
  int res = 0;
  for (int i = l; i <= r; i++) {
    res += cnt[i];
  }
  return res;
}

// 整体二分
void solve(int l, int r, vector<query> q) {
  int mid = l + r >> 2;
  if (l == r) {
    for (int i = 0; i < q.size(); i++) ans[q[i].id] = tmp[l];
    return;
  }
  vector<query> ql, qr;
  int t = check(l, mid);
  for (query i : q) {
    if (i.k <= t) ql.push_back(i);
    else i.k -= t, qr.push_back(i);
  }
  solve(l, mid, ql), solve(mid + 1, r, qr);
  return;
}

查询区间第 k 小

题 3 在一个数列中多次查询区间第 $k$ 小的数。

涉及到给定区间的查询，再按之前的方法进行二分就会导致 check 函数的时间复杂度爆炸。仍然考虑询问与值域中点 $m$ 的关系：若询问区间内小于等于 $m$ 的数有 $t$ 个，询问的是区间内的 $k$ 小数，则当 $k \leq t$ 时，答案应小于等于 $m$；否则，答案应大于 $m$．（注意边界问题）此处需记录一个区间小于等于指定数的数的数量，即单点加，求区间和，可用树状数组快速处理。为提高效率，只对数列中值在值域区间 $[l,r]$ 的数进行统计，即，在进一步递归之前，不仅将询问划分，将当前处理的数按值域范围划为两半。

#include <bits/stdc++.h>
using namespace std;

const int MAXN = 2e5 + 5;

struct num {
	int x, id;
};

struct query {
	int l, r, k, id;
};

int n, m;
int ans[MAXN];
vector<int> tmp;
int c[MAXN];

int lowbit(int x) {return x & -x;}

void add(int pos, int val) {
	while (pos <= n) {
		c[pos] += val;
		pos += lowbit(pos);
	}
}

int find(int x) {
	int res = 0;
	while (x) {
		res += c[x];
		x -= lowbit(x);
	}
	return res;
}

// zl、zr值域边界，q所有询问，a存储着原数组中值域在[zl,zr]的数字
void erfen(int zl, int zr, vector<query> q, vector<num> a) {
	if (zl == zr) {
		for (query i : q) ans[i.id] = tmp[zl];
		return;
	}
	int mid = zl + zr >> 1;
	vector<query> ql, qr;
	vector<num> al, ar;
	for (num i : a)
		if (i.x <= mid) al.push_back(i), add(i.id, 1);
		else ar.push_back(i);
	for (query &i : q) {
		int t = find(i.r) - find(i.l-1);  // 在树状数组中查询
		if (i.k <= t) ql.push_back(i);
		else i.k -= t, qr.push_back(i);
	}
	for (num i : al) add(i.id, -1);  // 清空树状数组
	erfen(zl, mid, ql, al);
	erfen(mid+1, zr, qr, ar);
}

int main() {
	cin >> n >> m;
	vector<num> A;
	for (int i = 1, x; i <= n; i++) {
		cin >> x;
		tmp.push_back(x);
		A.push_back({x, i});
	}
	vector<query> q;
	for (int i = 0, l, r, k; i < m; i++) {
		cin >> l >> r >> k;
		q.push_back({l, r, k, i});
	}
	// 离散化
	sort(tmp.begin(), tmp.end());
	tmp.erase(unique(tmp.begin(), tmp.end()), tmp.end());
	for (num &i : A) i.x = lower_bound(tmp.begin(), tmp.end(), i.x) - tmp.begin();
	erfen(0, tmp.size()-1, q, A);
	for (int i = 0; i < m; i++) cout << ans[i] << endl;
	return 0;
}

带修改操作的区间第 k 小

题 4 P2617 Dynamic Rankings - 洛谷 给定一个数列，要支持单点修改，区间查第 $k$ 小。

修改操作可以直接理解为从原数列中删去一个数再添加一个数，为方便起见，将询问和修改统称为「操作」。因后面的操作会依附于之前的操作，不能如题 3 一样将统计和处理询问分开，故可将所有操作存于一个数组，用标识区分类型，依次处理每个操作。为便于处理树状数组，修改操作可分拆为删除操作和插入操作。

优化

  1. 树状数组每次清空会导致时间复杂度爆炸，可采用每次使用树状数组时记录当前修改位置（这已由 1 中提到的临时数组实现），本次操作结束后在原位置加 $-1$ 的方法快速清零．
  2. 一开始对于数列的初始化操作可简化为插入操作．

#include <bits/stdc++.h>  
using namespace std;  
  
// 带有修改操作的多次查询区间第 k 小  
const int MAXN = 1e5 + 5;  
  
struct opt {  
    int t, l, r, k, id;  
    // t：操作类型，0：询问，1：插入，2：删除  
    // 插入：l：插入位置，k：插入的值  
    // 删除：l：删除位置，k：删除的值  
};  
  
int n, m;  
int a[MAXN];      // 辅助输入  
int ans[MAXN];  
vector<int> tmp;  // 离散化数组  
int c[MAXN];      // 树状数组  
  
int lowbit(int x) {  
    return x & -x;  
}  
  
int find(int x) {  
    int res = 0;  
    while (x) {  
        res += c[x];  
        x -= lowbit(x);  
    }  
    return res;  
}  
  
void add(int x, int k) {  
    while (x <= n) {  
        c[x] += k;  
        x += lowbit(x);  
    }  
}  
  
void erfen(int zl, int zr, vector<opt> q) {  
    if (zl == zr) {  
        for (opt i : q) if (i.t == 0) ans[i.id] = tmp[zl];  
        return;  
    }  
    int mid = zl + zr >> 1;  
    vector<opt> ql, qr;  
    for (opt &i : q) {  
        if (i.t == 0) {  // 询问  
            int s = find(i.r) - find(i.l-1);  
            if (i.k <= s) ql.push_back(i);  
            else i.k -= s, qr.push_back(i);  
        } else if (i.t == 1) {  // 插入  
            if (i.k <= mid) ql.push_back(i), add(i.l, 1);  
            else qr.push_back(i);  
        } else {  // 删除  
            if (i.k <= mid) ql.push_back(i), add(i.l, -1);  
            else qr.push_back(i);  
        }  
    }  
    // 清空树状数组  
    for (opt i : ql)  
        if (i.t == 1) add(i.l, -1);  
        else if (i.t == 2) add(i.l, 1);  
    erfen(zl, mid, ql);  
    erfen(mid+1, zr, qr);  
}  
  
int main() {  
    vector<opt> q;  
    cin >> n >> m;  
    for (int i = 1; i <= n; i++) {  
        cin >> a[i];  
        tmp.push_back(a[i]);  
        q.push_back({1, i, 0, a[i], 0});  
    }  
    // 操作的输入  
    int cnt = 0;  // 统计询问个数  
    while(m--) {  
        char op;  
        int l, r, k;  
        cin >> op;  
        if (op == 'Q') {  
            cin >> l >> r >> k;  
            q.push_back({0, l, r, k, cnt++});  
        } else {  
            cin >> l >> k;  
            tmp.push_back(k);  
            q.push_back({2, l, 0, a[l], 0});  
            q.push_back({1, l, 0, k, 0});  
            a[l] = k;  
        }  
    }  
    // 离散化  
    sort(tmp.begin(), tmp.end());  
    tmp.erase(unique(tmp.begin(), tmp.end()), tmp.end());  
    for (opt &i : q) if (i.t) i.k = lower_bound(tmp.begin(), tmp.end(), i.k) - tmp.begin();  
    erfen(0, tmp.size()-1, q);  
    for (int i = 0; i < cnt; i++) cout << ans[i] << endl;  
    return 0;  
}
